Odin Recipes

Welcome to Odin Recipies, a small recipe project from the Odin Project.

This project implements 2 parts that we're tested on me:

  • HTML Elements such as:
    • Headings and paragraphs
    • Lists
    • Links and Images
  • Basic CSS such as:
    • Basic CSS properties(i.e: Color, Fonts, Text-align)
    • The box model
    • Classes, ID's, Selectors
    • Padding and Margins

The purpose of this project is to create a simple webpage using fundamentals learnt from the Odin Project's Lessons and you'll see that most concepts were applied when needed
